# v0.3.1

Patch release of `googleart-download`.

## Fixes

- httpx client: ensure `proxy` parameter is used (not `proxies`) and `trust_env` handled consistently with 0.28 docs.
- Type check: satisfy mypy for optional `psutil` import in memory probe without changing runtime logic.
- Editor diagnostics: tighten `os.sysconf` detection via `getattr`+`callable` to remove attr warnings while keeping Unix support.
- Spellcheck: switch to `cspell:words` directive for `sysconf`/`AVPHYS` in image writer.

## Behavior improvements

- Safer AUTO backend selection for very large images when available memory is unknown: prefer BigTIFF to avoid OOM; small canvases remain Pillow.
- When BigTIFF is selected, final output extension is `.tif`. Pillow path remains `.jpg`.

## Notes

- BigTIFF/pyvips backends do not yet write EXIF; use `--write-sidecar` or force Pillow for smaller images.
- Python baseline is 3.12 (project and CI).
