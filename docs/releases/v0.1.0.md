# v0.1.0

First public release of `googleart-download`.

## Highlights

- Download high-resolution Google Arts & Culture artwork images by stitching tiles locally.
- Accept multiple input forms:
  - full artwork URL
  - artwork URL with viewer query params
  - official `g.co/arts/...` short links
  - bare asset ids
- Resume interrupted work through tile cache reuse and batch state persistence.
- Inspect available sizes before downloading, including raw canvas estimates.
- Handle very large artworks safely by switching to TIFF/BigTIFF streaming output.
- Export artwork metadata with `--metadata-only`, JSON sidecars, and optional JPEG EXIF output.
- Support batch resume, rerun of failed tasks, and explicit output conflict policies.

## Notes

- Normal-sized outputs default to JPEG.
- Very large outputs automatically switch to TIFF/BigTIFF.
- Automatic TIFF-to-JPEG conversion is intentionally not part of the default path.
